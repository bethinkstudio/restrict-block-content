Applies Restrict Content Pro level based restrictions to specific core blocks.

== Description ==

This plugin will allow you, by default, to add restrictions to `core/group` and `core/row` blocks in the block and full-site editors.

Restrictions involve enabling it for that block, then setting the access level used by RCP, and then determine whether the user must have at least that access level, or not have that access level in the block editor.
